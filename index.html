Elementos Semànticos

abbr
El elemento HTML abbr (o Elemento de Abreviación HTML) representa una abreviación o acrónimo; el atributo opcional title puede ampliar o describir la abreviatura. Si está presente, el atributo title debe contener la descripción completa y nada más.

U
Muestra el texto subrayado.
ejemplo: p Texto normal y... u Texto subrayado /u p
i
Muestra el texto marcado con un estilo en cursiva o italica.
Ejemplo: p Texto normal y... i Texto 'inclinado' i /p
B
Indica que el texto debe ser representado con una variable bold, o negrita, de la tipografía que se esté usando.
p Texto normal y... b Texto en negrita /b /p
Br
El elemento HTML line break br produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección, donde la división de las líneas es significante.
ejemplo: Mozilla Foundation br ,1981 Landings Drive br
Cite
Marca una referencia a una fuente, o el autor de un texto citado.
ejemplo: p cite Galileo /cite dijo: "... y sin embargo, se mueve." /p
Strong
El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto. ejemplo: p em El dinero /em es importante pero strong la salud /strong lo es más. /p
S
Muestra el texto tachado con una linea horizontal.
ejemplo: Texto normal y... s Texto tachado /s
Sub
El elemento HTML sub define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice. ejemplo:La fórmula química de agua: H sub 2/sub O
Sup
El elemento HTML define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice. ejemplo Este texto es sup superíndice /sup
Span
span - abarcar. Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea. ejemplo div style="border: 1px dotted blue;" h4 Ejemplo de div y span /h4 p Esto es un párrafo dentro de un div, span style="color: red;" y esto un span dentro de un párrafo. /span /p /div
time
El elemento HTML representa un periodo específico en el tiempo. Puede incluir el atributo datetime para convertir las fechas en un formato interno legible por un ordenador, permitiendo mejores resultados en los motores de búsqueda o características personalizadas como recordatorios.
Ejemplo: p El concierto empieza a las time 20:00 /time. /p
Code
Es el apropiado para marcar el código de un programa.
Ejemplo: code #include main() { for(;;) { printf ("Hello World!\n"); } } /code
a
El Elemento HTML Anchor crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.
Ejemplo: anclaje a un archivo externo a href="https://www.mozilla.com/" Enlace externo /a
