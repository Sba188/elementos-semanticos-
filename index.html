Elementos Semànticos

abbr
El elemento HTML (o Elemento de Abreviación HTML) representa una abreviación o acrónimo; el atributo opcional title puede ampliar o describir la abreviatura. Si está presente, el atributo title debe contener la descripción completa y nada más.
u
Muestra el texto subrayado.ejemplo: p Texto normal y... u Texto subrayado /u p
i
Muestra el texto marcado con un estilo en cursiva o italica. ejemplo: p Texto normal y... i Texto 'inclinado' i /p
b
Indica que el texto debe ser representado con una variable bold, o negrita, de la tipografía que se esté usando. p Texto normal y... b Texto en negrita /b /p
br
El elemento HTML line break br produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección, donde la división de las líneas es significante. ejemplo: Mozilla Foundation br ,1981 Landings Drive br
cite
Marca una referencia a una fuente, o el autor de un texto citado. ejemplo:p cite Galileo /cite dijo: "... y sin embargo, se mueve." /p
strong
El elemento strong es el apropiado para marcar con especial énfasis las partes más importantes de un texto. ejemplo: p em El dinero /em es importante pero strong la salud /strong lo es más. /p
s
Muestra el texto tachado con una linea horizontal. ejemplo: Texto normal y... s Texto tachado /s
sub
El elemento HTML sub define un fragmento de texto que se debe mostrar, por razones tipográficas, más bajo, y generalmente más pequeño, que el tramo principal del texto, es decir, en subíndice. ejemplo:La fórmula química de agua: H sub 2/sub O
sup
El elemento HTML define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño, que el tramo principal del texto, es decir, en superíndice. ejemplo Este texto es sup superíndice /sup
span
span - abarcar. Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea. ejemplo div style="border: 1px dotted blue;" h4 Ejemplo de div y span /h4 p Esto es un párrafo dentro de un div, span style="color: red;" y esto un span dentro de un párrafo. /span /p /div
